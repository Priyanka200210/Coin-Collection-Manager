# Coin Collection Manager - Java(Console Based)
 A Java-based application designed to help collectors manage their coin collections efficiently.
 This application provides comprehensive features for **adding, searching, and persisting coin data with database integration**.

 ## Features
 ### 1. Add Coins
**Individual Addition:** Add coins one at a time with detailed information 

**Bulk Upload:** Import multiple coins from a file   

**Coin Details Tracked:**   
Country of origin     
Denomination     
Year of minting     
Current market value     
Acquired date     

### 2. Search & Filter
**Create Lists by:**  
Country  
Year of Minting  
Current Value (with sorting capability)  

**Search Specific Coins by:**  
Country + Denomination  
Country + Year of Minting   
Country + Denomination + Year of Minting  
Acquired Date + Country  

### 3. Update Coins
**Modify Existing Coins:** Update coin details such as current value  
**Find and Edit:** Search for a coin and update its attributes  

### 4. Delete Coins
**Remove Single Coin:** Delete a specific coin from the collection  
**Selective Deletion:** Remove coins based on search criteria    

### 5. Data Persistence
Automatic loading of existing collection on startup  
Save current collection state to database  
Reliable data storage and retrieval  

## Technologies Used
**Language:** Java  
**Database:** MySQL  
**Collections Framework:** Java Collections (ArrayList, TreeSet, etc.) for in-memory data management  
**JDBC:** For database connectivity  
**IDE:** Eclipse
